# HW4: Stereo Matching
## Discriptions
* Disparity Estimation
## Usage
* Run `main.py` to generate disparity images.  
  ```
  python main.py
  ```
* (Optional) Generate all the intermediate images.  
  ```
  python main.py -v
  ```
* Run `eval_stereo.py` to evalute the bad pixel ratio.  
  ```
  python eval_stereo.py
  ```
